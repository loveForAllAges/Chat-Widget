Account detail

Чат-виджет - сервис для создания коммуникации между клиентами и операторами. Виждет позволяет общаться не только авторизованным пользователям, но и анонимным.

Django

Сервис реализован на Django. Сервис легко внедряется в другие проекты.

Django REST Framework

Сервис имеет некоторый функционал, который работает через API. Это необходимо для связи JavaScript и Django.

Django Channels

Сервис включает в себя библиотеку channels для реализации WebSocket.

Разграничение доступа

Сервис имеет 3 уровня доступа: Клиент, Оператор, Администратор.

Быстрота ответа

Сервис имеет таблицу с диалогами пользователей, в которой операторы принимают диалоги и ведут общение с клиентом.

Управление

Диалог с клиентом имеет 3 категории: ожидание, активный, завершен. Это позволяет удобно управлять операторами и быстро отвечать клиентам.

История

История диалога хранится до тех пор, пока жива сессия браузера пользователя. Для авторизованных пользователей история диалога не имеет ограничений.

Статистика

Сервис позволяет оценивать продуктивность сотрудников, благодаря таблице сотрудников.
